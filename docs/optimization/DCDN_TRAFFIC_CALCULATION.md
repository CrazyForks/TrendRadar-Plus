# DCDN 流量和费用计算

## 核心概念

- **PV**（Page View）：页面浏览量，用户每打开/刷新一个页面 = 1 PV
- **UV**（Unique Visitor）：独立访客，同一用户一天内 = 1 UV
- **请求数**：1 个 PV 会触发多个 HTTP 请求（HTML + CSS + JS + API + 图片），通常 20-30 个

## 费用计算公式

```
日均动态请求 = 日均 PV × 每页请求数 × API 占比 × (1 - 缓存命中率)
月度费用 = 日均动态请求 × 30 ÷ 1000 × 单价
```

**典型参数**：
- 每页请求数：~25
- API 占比：~35%
- DCDN 单价：0.01 元/千次

## 查看实际流量

```bash
# 服务器日志统计
ssh root@YOUR_SERVER_IP -p YOUR_SSH_PORT
docker exec hotnews cat /var/log/nginx/access.log | wc -l

# 缓存统计 API
curl http://your-domain.com/api/cache/stats
```
