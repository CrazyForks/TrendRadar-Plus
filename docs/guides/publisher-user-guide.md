# 多平台发布系统 - 用户指南

> 一键发布文章到知乎、掘金、CSDN、微信公众号等多个平台

## 功能概览

- 📝 富文本编辑器，支持标题、段落、列表、代码块、图片等
- 🖼️ 封面图片上传和裁剪（自动适配微信公众号比例）
- 📄 支持导入 Markdown、PDF、Word 文档
- 🔗 支持从 URL 导入网页内容
- 🤖 AI 润色功能（改写、扩写、缩写、翻译）
- 🚀 一键发布到多个平台
- 📊 发布历史记录和重试功能

## 快速开始

### 1. 安装浏览器插件

发布功能需要配合 HotNews 浏览器插件使用：

1. 打开 Chrome 网上应用店
2. 搜索 "HotNews Summarizer"
3. 点击"添加到 Chrome"

### 2. 登录各平台

发布前需要先登录目标平台：

- **知乎**: 访问 zhihu.com 并登录
- **掘金**: 访问 juejin.cn 并登录
- **CSDN**: 访问 csdn.net 并登录
- **微信公众号**: 访问 mp.weixin.qq.com 并登录

### 3. 创建文章

访问编辑器页面 `/write`，开始创作：

1. 输入标题（必填，最多 64 字）
2. 输入摘要（可选，最多 200 字）
3. 上传封面图片（可选，建议 900x383 像素）
4. 编写正文内容

### 4. 发布文章

1. 点击右上角「发布」按钮
2. 选择要发布的平台
3. 确认各平台登录状态（绿色表示已登录）
4. 点击「确认发布」

## 编辑器功能

### 工具栏

| 按钮 | 功能 | 快捷键 |
|------|------|--------|
| B | 加粗 | Ctrl+B |
| I | 斜体 | Ctrl+I |
| H1/H2/H3 | 标题 | - |
| 列表 | 无序列表 | - |
| 数字列表 | 有序列表 | - |
| 引用 | 引用块 | - |
| 代码 | 代码块 | - |
| 链接 | 插入链接 | - |
| 图片 | 上传图片 | - |

### 自动保存

- 编辑器每 30 秒自动保存一次
- 同时保存到服务器和本地浏览器
- 意外关闭页面后可恢复内容

### 封面裁剪

1. 点击封面区域上传图片
2. 拖动裁剪框调整位置
3. 点击「确认裁剪」
4. 裁剪后的图片自动适配微信公众号封面比例（2.35:1）

## 导入功能

### 导入文档

支持导入以下格式：

- **Markdown** (.md) - 保留格式
- **PDF** (.pdf) - 提取文本
- **Word** (.docx) - 保留基本格式

操作步骤：
1. 点击工具栏「导入」按钮
2. 拖拽文件到上传区域，或点击选择文件
3. 等待解析完成

### 从 URL 导入

1. 在浏览器中打开要导入的网页
2. 右键点击页面
3. 选择「将此页面发送到编辑器」
4. 自动跳转到编辑器并导入内容

### 从选中文字导入

1. 在网页中选中要导入的文字
2. 右键点击选中区域
3. 选择「发送到 HotNews 编辑器」
4. 自动跳转到编辑器并导入内容

## AI 润色

选中编辑器中的文字后，会出现 AI 润色菜单：

| 功能 | 说明 |
|------|------|
| ✏️ 改写 | 保持原意，换种表达方式 |
| 📝 扩写 | 添加更多细节和内容 |
| 📋 缩写 | 精简内容，保留核心 |
| 🌐 翻译 | 翻译成其他语言 |

## 发布流程

### 平台登录检测

发布前会自动检测各平台的登录状态：

- 🟢 绿色：已登录，可以发布
- 🔴 红色：未登录，点击可跳转登录页

### 发布结果

发布完成后会显示每个平台的结果：

- ✅ 成功：显示文章链接，可点击查看
- ❌ 失败：显示错误原因，可点击重试

### 重试失败项

如果部分平台发布失败：

1. 点击失败项旁边的「重试」按钮
2. 或点击「重试全部失败项」批量重试

## 发布历史

访问 `/publish-history` 查看所有发布记录：

- 按平台筛选
- 按状态筛选（成功/失败）
- 点击可跳转到原文章或草稿

## 常见问题

### Q: 发布时提示"插件未安装"？

A: 请确保已安装 HotNews 浏览器插件，并刷新页面重试。

### Q: 发布时提示"未登录"？

A: 请先在新标签页登录对应平台，然后点击「刷新登录状态」按钮。

### Q: 微信公众号发布后图片不显示？

A: 微信公众号不支持外部图片链接，发布后需要在微信后台手动上传图片。

### Q: 发布超时怎么办？

A: 可能是网络问题或平台响应慢，请稍后点击「重试」按钮。

### Q: 草稿丢失了怎么办？

A: 编辑器会自动保存到本地浏览器，重新打开编辑器页面会提示恢复。
