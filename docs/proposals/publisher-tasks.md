## 依赖关系

```
M1-01 (数据库) ──┬── M1-02 (草稿API)
                │
                ├── M1-03 (图片API)
                │
                └── M1-04 (导入API)

M1-06 (页面框架) ── M1-07 (Tiptap) ── M1-09 (自动保存)

M2-01 (通信桥) ── M2-03 (发布管理) ──┬── M2-05 (知乎)
                                    ├── M2-10 (掘金)
                                    ├── M2-11 (CSDN)
                                    └── M3-02 (微信)
```

## 风险任务

| 任务 | 风险 | 应对 |
|------|------|------|
| M2-05 知乎适配器 | 知乎改版导致失效 | 参考 MultiPost 实现，做好错误处理 |
| M3-02 微信适配器 | API 逆向复杂 | 预留更多时间，准备 DOM 备选方案 |
| M3-05 PDF 解析 | 复杂 PDF 解析不完整 | 只支持简单文本 PDF，复杂的提示用户 |
