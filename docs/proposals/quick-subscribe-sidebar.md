# 快速订阅侧边栏方案

## 背景

当前用户点击设置按钮后会跳转到独立的设置页面，需要离开当前浏览的新闻列表。这增加了订阅标签/源的操作成本，降低了用户订阅的意愿。

## 目标

在主页面点击设置按钮时，从右侧滑出一个侧边栏面板，直接显示"为你推荐 | 标签 | 订阅源 | 关键词 | 公众号"内容，让用户无需离开当前页面即可快速订阅。

## 设计方案

### 1. 交互流程

```
用户点击 ⚙️ 设置按钮
    ↓
检查登录状态
    ↓ (未登录)
弹出登录框
    ↓ (已登录)
从右侧滑出订阅侧边栏
    ↓
显示 Tab 切换：为你推荐 | 标签 | 订阅源 | 关键词 | 公众号
    ↓
用户可直接点击 "+关注" 订阅
    ↓
点击遮罩层或关闭按钮关闭侧边栏
```

### 3. 功能模块

#### 3.1 为你推荐 Tab
- 调用 `/api/user/preferences/recommended-tags` 获取推荐
- 显示热门标签、新发现标签、相关标签
- 每项显示图标、名称、描述、+关注按钮

#### 3.2 标签 Tab
- 调用 `/api/user/preferences/tags` 获取所有标签
- 支持搜索过滤
- 支持分类筛选：全部 | 热门 | 新发现 | 大类 | 主题 | 属性
- 每项显示图标、名称、+关注按钮

#### 3.3 订阅源 Tab
- 搜索框输入关键词搜索
- 调用 `/api/user/preferences/search-sources` 搜索
- 显示源名称、URL、+订阅按钮

#### 3.4 关键词 Tab
- 输入框添加关键词
- 调用 `/api/user/keywords` 管理
- 显示已添加的关键词列表

#### 3.5 公众号 Tab
- 需要微信授权
- 搜索公众号
- 显示已订阅的公众号列表

### 4. 技术实现

#### 4.1 新增文件
- `hotnews/hotnews/web/static/js/src/subscribe-sidebar.js` - 侧边栏 JS 模块

#### 4.2 修改文件
- `hotnews/hotnews/web/static/css/viewer.css` - 添加侧边栏样式
- `hotnews/hotnews/web/static/js/src/init.js` - 修改 `goToSettings()` 函数

#### 4.3 样式复用
- 复用 `.todo-sidebar` 的基础框架样式
- 复用 `user_settings.css` 中的标签卡片样式

#### 4.4 API 复用
现有 API 已足够，无需新增：
- `GET /api/user/preferences/recommended-tags` - 获取推荐
- `GET /api/user/preferences/tags` - 获取所有标签
- `POST /api/user/preferences/tags` - 关注标签
- `DELETE /api/user/preferences/tags/{tag_id}` - 取消关注
- `GET /api/user/preferences/search-sources` - 搜索订阅源
- `POST /api/user/preferences/sources` - 订阅源
- `GET/POST/DELETE /api/user/keywords` - 关键词管理

### 5. 与设置页面的关系

- 侧边栏是**轻量级快速订阅入口**，专注于"添加订阅"
- 设置页面保留，用于**完整的偏好管理**：
  - 查看偏好统计
  - 管理已关注列表（拖拽排序、取消关注）
  - 栏目设置
  - 更多高级设置

- 侧边栏 Header 添加"去设置"链接，方便用户进入完整设置页面

### 6. 移动端适配

- 宽度：100%（全屏）
- 高度：100vh
- 底部安全区域适配

### 7. 实现步骤

1. **Phase 1**: 创建侧边栏框架

   - CSS 样式（复用 todo-sidebar）
   - 打开/关闭逻辑

2. **Phase 2**: 实现"为你推荐" Tab
   - 加载推荐数据
   - 渲染推荐列表
   - 关注/取消关注功能

3. **Phase 3**: 实现"标签" Tab
   - 搜索功能
   - 分类筛选
   - 关注功能

4. **Phase 4**: 实现"订阅源" Tab
   - 搜索功能
   - 订阅功能

5. **Phase 5**: 实现"关键词" Tab
   - 添加关键词
   - 删除关键词

6. **Phase 6**: 实现"公众号" Tab
   - 微信授权检查
   - 搜索订阅功能
