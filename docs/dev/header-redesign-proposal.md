# Header 重构方案：毛玻璃一体化导航

> 预览文件：`header-preview.html`（同目录下，浏览器打开可交互）

## 现状问题

1. GitHub/Gitee 图标占据 header 左侧，对普通用户无意义
2. 主题 tab 和普通栏目混在一起，栏目多时拥挤
3. header 和 tabs 是两个独立白色卡片，浪费垂直空间
4. 栏目 tab 选中态用背景色填充，视觉较重

## 方案概述

将 header 和 tabs 合并为一个毛玻璃卡片，分两层：
- 上层：主导航（主页 / 我的主题）+ 工具按钮
- 下层：子栏目（根据主导航切换内容）

整个卡片 sticky 吸顶，滚动时透出背景内容。

## 布局结构

```
┌─────────────────────────────────────────────────────────────┐
│  [ 主页 ]  [ 我的主题 ]                   ⭐ 🌙 🔍 ➕ 👤  │  ← 主导航 + 工具栏
│─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─│  ← 细分隔线
│  我的关注   新发现   精选博客   AI早报   公众号   财经投资    │  ← 子栏目（纯文字）
│            ───────                                          │  ← 滑动底线指示器
└─────────────────────────────────────────────────────────────┘
```

切换到"我的主题"时，子栏目变为：

```
│  + 新建主题   深圳安居人才房   书籍推荐   google   Anthropic  │
│                ──────────────                                │
```

## 设计细节

### 毛玻璃卡片
- `background: rgba(255, 255, 255, 0.72)`
- `backdrop-filter: saturate(180%) blur(20px)`
- `border: 1px solid rgba(0, 0, 0, 0.06)`
- `border-radius: 14px`
- `position: sticky; top: 12px`

### 主导航按钮
- 纯文字，无 emoji
- 未选中：`color: #86868b`，无背景
- Hover：`color: #1d1d1f`，`background: rgba(0,0,0,0.05)`
- 选中：`color: #1d1d1f`，`background: rgba(0,0,0,0.07)`
- `font-size: 14px`，`font-weight: 600`，`border-radius: 8px`

### 子栏目
- 纯文字，无 emoji，无背景色填充
- 未选中：`color: #86868b`，`font-weight: 500`
- 选中：`color: #1d1d1f`，`font-weight: 600`
- 选中指示器：底部 2px 线，`background: #1d1d1f`
- 指示器带滑动动画：`transition: 0.3s cubic-bezier(0.4, 0, 0.2, 1)`
- `font-size: 13px`

### 工具按钮
- 中性灰调：`background: rgba(0,0,0,0.04)`，`color: #6b7280`
- Hover：`background: rgba(0,0,0,0.08)`
- 30x30px，`border-radius: 8px`

### 分隔线
- 主导航和子栏目之间：`0.5px solid rgba(0,0,0,0.08)`

### "+ 新建主题"
- 与其他子栏目样式完全一致，不做特殊突出

## 状态处理

| 状态 | 主导航 | 子栏目 |
|------|--------|--------|
| 已登录 - 主页 | 主页(选中) / 我的主题 | 普通栏目列表 |
| 已登录 - 我的主题 | 主页 / 我的主题(选中) | 主题列表 + 新建主题 |
| 未登录 | 主页(选中) / 我的主题 | 普通栏目列表 |
| 未登录点击"我的主题" | 弹出登录弹窗 | 不切换 |

## 涉及文件

| 文件 | 改动 |
|------|------|
| `viewer.html` | header HTML 重构：去掉 GitHub/Gitee 链接，改为主导航结构 |
| `viewer.css` | 毛玻璃样式、子栏目底线指示器、去掉旧 tab 背景色 |
| `tabs.js` | 主导航切换逻辑、子栏目滑动指示器、显示/隐藏对应 tab 组 |
| `topic-tracker.js` | 主题 tabs 渲染适配新结构（渲染到子栏目区域） |
| `init.js` | 主导航初始化、状态恢复 |

## 不变的部分

- 右侧工具按钮功能不变（收藏、护眼、搜索、设置、登录）
- 内容区域渲染逻辑不变
- 后端 API 不变
- 子栏目内的 emoji 图标去掉，但卡片内容区的 emoji 保留
