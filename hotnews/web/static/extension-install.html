uihash 总结助手

一键 AI 总结，让阅读更高效

📥 下载插件 v1.3.1

适用于 Chrome、Edge、Arc 等 Chromium 内核浏览器

📋 安装步骤

1
下载并解压
点击上方按钮下载 ZIP 文件，解压到一个固定位置（如桌面或文档文件夹）。
注意：解压后请勿删除或移动该文件夹。
2
打开扩展管理页面
在浏览器地址栏输入 chrome://extensions 并回车。
或点击浏览器右上角菜单 → 扩展程序 → 管理扩展程序。
3
开启开发者模式
在扩展管理页面右上角，打开「开发者模式」开关。
4
加载已解压的扩展
点击「加载已解压的扩展程序」按钮，选择刚才解压的文件夹。
5
固定到工具栏（推荐）
点击浏览器右上角的拼图图标 🧩，找到「uihash 总结助手」，点击📌固定。
💡 提示：安装完成后，访问任意网页时会看到右下角的浮动按钮。点击即可打开侧边栏进行 AI 总结。

✨ 功能特点

🤖
智能总结 一键提取文章核心内容，支持多种文章类型
📋
行动清单 自动提取待办事项，一键添加到 Todo
💬
智能问答 基于文章内容进行深度对话
🔄
多端同步 登录后数据自动同步
