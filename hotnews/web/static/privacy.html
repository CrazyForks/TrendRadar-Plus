🔒 隐私政策

最后更新日期：2026年1月27日

概述

uihash 总结助手（以下简称"本扩展"）是一款浏览器扩展程序，旨在帮助用户快速获取网页内容的 AI 智能总结。我们非常重视用户隐私，本隐私政策说明了我们如何收集、使用和保护您的信息。

信息收集

我们收集的信息

  1. 页面内容数据 - 当您主动点击"总结"按钮时，我们会提取当前页面的文本内容。此数据仅用于生成 AI 总结，不会永久存储在我们的服务器上。
  2. 账户信息 - 如果您选择登录，我们会收集您的基本账户信息（昵称、头像）。我们不会存储您的第三方登录密码。
  3. 使用数据 - 总结次数统计、收藏的文章和总结、Todo 待办事项。
  4. 本地存储数据 - 用户偏好设置、浮动按钮位置、聊天历史记录（存储在本地浏览器中）。
我们不收集的信息：
  • 我们不会收集您的浏览历史
  • 我们不会在后台自动抓取任何页面内容
  • 我们不会收集您的个人身份信息（除非您主动登录）

信息使用

我们收集的信息仅用于以下目的：

  1. 提供核心服务 - 生成网页内容的 AI 总结
  2. 账户管理 - 管理您的登录状态和使用配额
  3. 功能改进 - 分析使用模式以改进产品体验
  4. 客户支持 - 响应您的问题和反馈

信息共享

我们不会出售、交易或以其他方式向外部方转让您的个人信息，除非：

  • 获得您的明确同意
  • 法律要求披露
  • 保护我们的权利、财产或安全

第三方服务

本扩展使用以下第三方服务：

  • AI 服务提供商 - 用于生成内容总结（数据传输加密，不永久存储）
  • OAuth 登录提供商 - GitHub、Google、微信（仅获取基本账户信息）

数据安全

我们采取以下措施保护您的数据：

  1. 所有数据传输使用 HTTPS 加密
  2. 服务器数据加密存储
  3. 定期安全审计
  4. 最小权限原则

权限说明

本扩展请求以下浏览器权限：

权限 用途
activeTab 获取当前标签页内容用于总结
storage 本地存储用户设置和数据
sidePanel 显示侧边栏界面
tabs 获取页面标题和 URL
cookies 管理登录状态
scripting 注入内容提取脚本
host_permissions 在任意网页上运行（用于提取内容）

您的权利

您有权：

  1. 访问 - 查看我们收集的关于您的信息
  2. 更正 - 更新您的账户信息
  3. 删除 - 请求删除您的账户和相关数据
  4. 导出 - 导出您的收藏和 Todo 数据
  5. 退出 - 随时停止使用本扩展

儿童隐私

本扩展不面向 13 岁以下的儿童。我们不会故意收集儿童的个人信息。

政策更新

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，并更新"最后更新日期"。重大变更时，我们会通过扩展内通知告知您。
