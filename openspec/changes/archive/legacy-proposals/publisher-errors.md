## 常见问题速查

| 问题类型 | 常见原因 | 快速解决 |
|----------|----------|----------|
| API 401 | Token 过期 | 重新登录 |
| API 403 | 非会员 | 检查会员状态 |
| 图片上传失败 | 格式/大小 | 检查限制 |
| 平台注入失败 | DOM 变化 | 更新选择器 |
| 发布超时 | 网络/平台慢 | 增加超时时间 |
