## ✅ 测试清单

### 1. 基本功能测试

#### 关注标签
- [ ] 点击任意标签的「关注」按钮

- [ ] 验证：加载完成后显示 ✓ 图标（1秒后消失）
- [ ] 验证：标签出现在「已关注的标签」区域
- [ ] 验证：统计数字「关注标签」+1

#### 屏蔽标签
- [ ] 点击任意标签的「屏蔽」按钮
- [ ] 验证：标签立即变红（< 50ms）
- [ ] 验证：按钮文字变为「✓ 已屏蔽」
- [ ] 验证：显示旋转加载图标 ⟳
- [ ] 验证：加载完成后显示 ✓ 图标（1秒后消失）
- [ ] 验证：标签出现在「已屏蔽的标签」区域
- [ ] 验证：统计数字「屏蔽标签」+1

#### 取消关注
- [ ] 点击已关注标签的「✓ 已关注」按钮
- [ ] 验证：标签立即恢复默认颜色
- [ ] 验证：按钮文字变回「关注」
- [ ] 验证：标签从「已关注的标签」区域消失
- [ ] 验证：统计数字「关注标签」-1

- [ ] 点击已屏蔽标签的「✓ 已屏蔽」按钮
- [ ] 验证：标签立即恢复默认颜色
- [ ] 验证：按钮文字变回「屏蔽」
- [ ] 验证：标签从「已屏蔽的标签」区域消失
- [ ] 验证：统计数字「屏蔽标签」-1

### 2. 视觉反馈测试

#### 加载状态
- [ ] 点击按钮后立即看到旋转图标 ⟳
- [ ] 旋转动画流畅（1秒/圈）
- [ ] 加载图标在按钮左侧显示

#### 成功状态
- [ ] 操作成功后显示 ✓ 图标
- [ ] ✓ 图标有淡入动画
- [ ] ✓ 图标 1 秒后自动消失

#### 颜色变化
- [ ] 关注：标签背景变为浅绿色，边框变为绿色
- [ ] 屏蔽：标签背景变为浅红色，边框变为红色
- [ ] 默认：标签背景为深色，边框为灰色
- [ ] 颜色过渡平滑（0.2秒）

#### 按钮状态
- [ ] 未激活：灰色文字，透明背景
- [ ] 激活（关注）：绿色文字，浅绿背景，绿色边框
- [ ] 激活（屏蔽）：红色文字，浅红背景，红色边框
- [ ] Hover：背景变亮

### 4. 错误处理测试

#### 网络失败模拟
**方法 1：浏览器开发者工具**
1. 打开 Chrome DevTools (F12)
2. 切换到 Network 标签
3. 选择 "Offline" 模式
4. 点击任意标签按钮
5. 验证：
   - [ ] UI 立即更新（乐观更新）
   - [ ] 显示加载图标
   - [ ] 几秒后自动回滚到原状态

   - [ ] Toast 3 秒后自动消失

**方法 2：断开网络**
1. 断开 WiFi/网络

## 💡 测试建议

### 重点测试
1. **乐观更新**：点击后立即看到变化
2. **双按钮**：关注和屏蔽操作清晰
3. **错误回滚**：网络失败时自动恢复

### 常见问题排查
1. **UI 没有立即更新**
   - 检查浏览器控制台是否有 JavaScript 错误
   - 清除浏览器缓存并刷新

2. **按钮点击无反应**
   - 检查是否处于 pending 状态（有加载图标）
   - 等待当前操作完成后再试

3. **状态不同步**
   - 刷新页面重新加载
   - 检查网络连接
