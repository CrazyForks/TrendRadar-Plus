# RSS 订阅源页签功能

## 📋 项目概述

为 hotnews 系统添加一个新的"订阅源"页签，允许用户搜索、预览和订阅后端配置的 RSS 源和自定义源，实现个性化的内容订阅体验。

## 🎯 核心功能

1. **源搜索**: 用户可以通过关键词搜索 RSS 源和自定义源
2. **源预览**: 点击源后显示最新文章列表和统计信息
3. **订阅管理**: 一键订阅/取消订阅感兴趣的源
4. **个性化内容**: 在订阅源页签只显示已订阅源的内容
5. **统计信息**: 显示订阅者数量、文章发布频率等

## 💡 设计亮点

### 1. 参考业界最佳实践
- 借鉴 Feedly、Inoreader 等成熟 RSS 阅读器的交互设计
- 简洁直观的搜索和订阅流程

### 2. 与现有系统无缝集成
- 复用现有的认证系统
- 利用现有的 RSS 源和自定义源数据
- 与标签系统并行，互不干扰

### 3. 性能优化
- 搜索结果缓存
- 预览内容缓存
- 分页加载
- Debounce 防抖

### 4. 用户体验优先
- 实时搜索反馈
- 乐观更新（订阅按钮立即响应）
- 友好的错误提示
- 响应式设计，支持移动端

## 📊 预期效果

- **提升用户粘性**: 用户可以订阅感兴趣的源，获得个性化内容
- **增加活跃度**: 订阅功能鼓励用户更频繁地访问
- **数据洞察**: 通过订阅数据了解用户兴趣偏好
- **推荐基础**: 为后续的智能推荐功能打下基础
